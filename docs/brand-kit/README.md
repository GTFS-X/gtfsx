# GTFS·X Brand Kit

Everything you need to use the GTFS·X identity in product, marketing, and docs.

## What's here

- **GTFS·X Brand Guidelines.html** — the full brand book. Open in any browser.
- **assets/brand/** — production SVG files. Pixel-tuned per favicon size.
- **brand-guidelines.css**, **colors_and_type.css**, **fonts/** — supporting files for the HTML doc.

## Quick start (favicon)

```html
<link rel="icon" type="image/svg+xml" href="/gtfsx-mark.svg">
<link rel="icon" type="image/svg+xml" sizes="16x16" href="/gtfsx-mark-16.svg">
<link rel="apple-touch-icon" href="/gtfsx-mark-256.svg">
```

## Brand name

Written form: **GTFS·X** (middle dot, U+00B7 — `Opt+Shift+9` on macOS). Not "GTFS X", "GTFSX", or "GTFS-X". Spoken: "G-T-F-S X". Domain: `gtfsx.dev`.

## Colors

- **Coral** — `#E8734A` (brand, primary action)
- **Cream** — `#FFF8F0` (canvas)
- **Dark Brown** — `#3D2E22` (ink)

Full palette in `colors_and_type.css`.

## Type

- **Nunito Black** — wordmark and headings (path-based in lockup SVGs — no font dependency for the logo)
- **Inter** — body & UI
- **JetBrains Mono** — times, IDs, code
